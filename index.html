Guide

Workflow

Start with Site to validate one design point, then move to CSV for regional scans, Array for farm-level anchoring, and Study for sensitivity runs.

Inputs

Load, angle, water depth, soil, mooring geometry, vessel limits, and cost assumptions drive the candidate ranking. Hover any option for field-level guidance.

CSV Columns

Use Latitude, Longitude, WaterDepth, Gravel, Sand, Mud, Clay, and FolkCde. Map scan filters by bounding box; nearest site selects the closest retained row.

Outputs

Results include best feasible anchor, variant, vessel, mass, holding capacity, mooring cost, total system cost, USD/kW, and optional LCOE contribution.

Public Use

The app runs fully in the browser. Uploaded CSV data is parsed locally in the page session and is not sent to a server by this static site.

Engineering Status

This is a screening-level JavaScript port of the MATLAB workflow. Final engineering release should compare golden MATLAB cases before certification.

Methodology & QA

Selection Logic

The selector evaluates gravity, driven pile, suction anchor, drag embedment, and vertical load anchor candidates against soil compatibility, load angle, holding capacity, geometry, and vessel constraints.

Cost Model

Per-device economics combine fabrication, installation vessel time, mooring line mass, exchange rate, number of lines, and rated power normalization to USD/kW.

Mooring & LCOE

Mooring material, minimum breaking load, line length, fixed charge rate, net capacity factor, and optional OpEx determine the mooring-plus-anchor LCOE contribution.

CSV Soil Mapping

Regional scans infer anchor soil class from Gravel, Sand, Mud, Clay, and FolkCde fields, then apply the same single-site selector row by row.

Quality Gates

Current automated checks cover soil classes, mooring materials, advanced overrides, CSV scan, array sharing, spacing failure, and parametric sweeps. Golden MATLAB regression remains the final certification step.

Responsible Use

Use results for screening, comparison, and research planning. Detailed design should include site investigation, geotechnical review, project-specific installation constraints, and independent engineering validation.
